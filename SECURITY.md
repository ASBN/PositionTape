# Security

Please do not report secrets in public issues. PositionTape is a diagnostic library; avoid adding telemetry, network calls or credentials to core packages.

## Reporting

For the alpha, report suspected vulnerabilities privately to the maintainers
through the repository owner's preferred private contact channel. If no private
channel is published yet, open a public issue that asks for a private security
contact without including exploit details, secrets, payload samples containing
private data, or credentials.

## Supported versions

No package versions are supported yet because PositionTape has not published a
release package. The `main` branch and documented alpha checkpoints are the only
security review targets.
