# Open Source and ASBN commercial boundary

## Open Source PositionTape

Allowed:

- Spec.
- Fixtures.
- Core algorithms.
- Language libraries.
- Basic logger adapters.
- Basic CLI.
- Test utilities.
- Conformance runner.

## ASBN commercial product

Keep out of this repo:

- Compiled GUI inspector.
- Enterprise importers.
- PDF/HTML evidence reports.
- License enforcement.
- Customer dashboards.
- Paid support automation.
- Cloud integrations requiring credentials.
- Proprietary heuristics for mutation/reordering visualization.

## Suggested commercial name

ASBN PositionTape Inspector.
