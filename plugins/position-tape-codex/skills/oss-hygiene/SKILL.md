# PositionTape OSS hygiene skill

Use this skill for project documentation, contribution flow, licensing, CI and release preparation.


Always obey AGENTS.md and docs/agentic-plan/APPROVALS_AND_PERMISSIONS.md. Stay inside the repository. Update AGENT_RUN_LOG.md after each checkpoint. Use fixtures as the source of truth.


## Required OSS files

- README.md
- LICENSE
- NOTICE if needed
- CONTRIBUTING.md
- CODE_OF_CONDUCT.md
- SECURITY.md
- CHANGELOG.md

## Release boundary

Prepare package metadata, but do not publish, push tags or create GitHub releases unless Alfonso explicitly asks.

## License policy

Default to Apache-2.0 unless Alfonso asks for MIT. Avoid GPL/AGPL dependencies unless explicitly approved.
